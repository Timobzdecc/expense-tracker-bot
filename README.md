# 💳 Expense Tracker Bot (AI-Powered)

Умный Telegram-бот для учёта личных финансов на базе **aiogram 3** и **Google Gemini AI**.
Бот автоматически распознает траты из текста и по фотографиям чеков, распределяет их по категориям, ведёт учёт бюджета и строит красивые диаграммы.

## ✨ Возможности

- 🤖 **AI-парсинг трат**: Просто напишите «бензин роснефть 500» или «500 бензин». Бот сам извлечёт сумму, описание и автоматически подберёт категорию с помощью Gemini AI.
- 📸 **Распознавание чеков**: Отправьте боту фотографию бумажного чека или скриншот банковского перевода. Gemini Vision распознает все позиции, извлечёт суммы и разнесёт их по категориям.
- 💬 **Режим чата с ИИ**: Полноценный встроенный чат-бот на базе Gemini с сохранением контекста диалога (безупречно отделяет логику рассуждений модели от финального ответа).
- 📉 **Диаграммы**: Генерация красивых столбчатых и круговых графиков (По категориям, По дням) прямо в чате с использованием Matplotlib.
- 💰 **Бюджеты**: Установка месячных лимитов на разные категории. Бот предупредит, если вы превысили лимит.
- 🔒 **Приватность**: Доступ к боту только для разрешенных пользователей (Whitelist), либо открытый доступ для всех желающих.
- 🗄 **Локальная БД**: Использование SQLite для хранения трат (база хранится локально на вашем устройстве, никакие сторонние базы данных не требуются).
- ⚙️ **Интерактивная настройка**: При первом запуске бот сам попросит ввести нужные ключи через консоль (без необходимости вручную копаться в конфигурационных файлах).

## 🚀 Установка и запуск (Для обычных пользователей)

Вам не нужно устанавливать Python или навыки программиста, чтобы запустить бота! Мы подготовили готовые независимые сборки.

### Для Windows (.exe)
1. Скачайте `ExpenseBot.exe` со страницы [Releases](https://github.com/Timobzdecc/expense-tracker-bot/releases).
2. Запустите файл в любой удобной папке. Откроется консоль, которая попросит вас ввести:
   - Токен Telegram-бота (получается у @BotFather).
   - API-ключ Google Gemini (от aistudio.google.com).
   - Ваш Telegram ID (или просто нажмите Enter, если хотите сделать бота публичным).
3. Бот автоматически создаст конфигурационный файл `.env` и базу данных рядом с собой и начнёт работу.

### Для Ubuntu / Debian (.deb)
1. Скачайте `.deb` пакет со страницы Releases.
2. Установите его командой:
   ```bash
   sudo apt install ./expensebot_1.0.0_amd64.deb
   ```
3. Приложение автоматически зарегистрируется как системная служба `expensebot`.

## 💻 Запуск из исходного кода (Для разработчиков)

1. Клонируйте репозиторий: `git clone https://github.com/Timobzdecc/expense-tracker-bot.git`
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите `python bot.py` — скрипт сам заботливо попросит ввести нужные ключи при первом запуске.

## 🌍 Развертывание на сервере (Docker)

Это самый простой и надёжный способ для серверов. База данных сохраняется благодаря привязанным volumes.

1. Клонируйте репозиторий на сервер.
2. Создайте файл `.env` в корне проекта и пропишите туда ключи (формат: `BOT_TOKEN=...`, `GEMINI_API_KEY=...`, `ALLOWED_USERS=...`).
3. Запустите контейнер в фоне:
   ```bash
   docker-compose up -d --build
   ```
*Для просмотра логов:* `docker-compose logs -f`
*Для остановки:* `docker-compose down`

## 🛠 Автоматическая сборка (CI/CD)

В репозиторий встроен **GitHub Actions**. При ручном запуске скрипта (через вкладку *Actions -> Build and Release -> Run workflow*) серверы GitHub автоматически:
1. Собирают `.exe` файл для Windows.
2. Собирают `.deb` пакет для Ubuntu Linux.
3. Загружают обе версии прямо в Релизы.

## 📖 Как пользоваться (Мануал)

- **Текстом:** Напишите сумму и описание. Порядок не важен. *Пример:* `Кофе старбакс 350` или `350 кофе`.
- **Фотографией:** Скиньте фото чека. Бот обработает его 10-20 секунд и запишет все купленные позиции отдельными тратами.
- **Ошибки AI:** Если нейросеть заблокирует запрос (сработает safety-фильтр), трата всё равно будет сохранена в категорию «🔧 Прочее», так как суммы и описания дополнительно парсятся надежным регулярным выражением.
- **Меню:**
  - **📊 Статистика**: Сводка за сегодня, неделю, месяц или всё время.
  - **📈 По дням**: Расходы за последние 7 дней в виде текстового графика.
  - **📉 Диаграммы**: Графики в виде картинок (круговая по категориям, столбчатая по дням).
  - **📋 История**: Просмотр и удаление последних 5 трат.
  - **💰 Бюджеты**: Установка лимита на категорию. Бот уведомит при превышении.
  - **💬 Чат с ИИ**: Переход в режим диалога с умным ассистентом (выход по кнопке «🚪 Выйти из чата»).

## 📱 Мобильное приложение (Android)

Помимо Telegram-бота, проект содержит полноценное нативное мобильное приложение для Android.

### Основные возможности приложения:
- 🎨 **Современный интерфейс**: Написан полностью на **Jetpack Compose** с использованием дизайн-системы Material 3.
- 📸 **Сканер чеков**: Вы можете сфотографировать бумажный чек или скриншот перевода через встроенную камеру — приложение отправит снимок в Gemini Vision, распознает все позиции и автоматически внесёт их в историю.
- 🔐 **Локальное хранилище**: База данных **Room** хранит все ваши траты локально на смартфоне.
- ⚙️ **Динамическая настройка**: API-ключ Gemini и выбор моделей настраиваются прямо на экране настроек приложения — никаких жёстко зашитых ключей в коде.
- 💬 **Интерактивный чат с ИИ**: Полноценный чат-помощник с **сохранением истории переписки** в локальную базу данных (история не пропадает при перезапуске).
- 🧠 **Фильтрация мыслей модели**: Встроенная система фильтрации автоматически вырезает из ответов ИИ любые служебные цепочки рассуждений, отдавая пользователю только готовый к чтению текст.
- 🏷 **Автокатегоризация**: Интеллектуальный разбор трат (например, при вводе «пиво 100» приложение корректно запишет трату в категорию «Еда» благодаря очистке мета-данных перед валидацией).

### Как установить:
1. Перейдите в раздел **Releases** на главной странице репозитория.
2. Найдите последний релиз Android-версии и скачайте файл **`app-release.apk`**.
3. Запустите APK-файл на своём смартфоне и подтвердите установку. 
   *(Сборки автоматически подписываются отладочным сертификатом на сервере GitHub, что позволяет устанавливать новые версии поверх старых без необходимости удалять приложение и терять историю финансов)*.
