# 💳 Expense Tracker Bot (AI-Powered)

Умный Telegram-бот для учёта личных финансов на базе **aiogram 3** и **Google Gemini AI**.
Бот автоматически распознает траты из текста и по фотографиям чеков, распределяет их по категориям, ведёт учёт бюджета и строит красивые диаграммы.

## ✨ Возможности

- 🤖 **AI-парсинг трат**: Просто напишите «бензин роснефть 500» или «500 бензин». Бот сам извлечёт сумму, описание и автоматически подберёт категорию с помощью Gemini AI.
- 📸 **Распознавание чеков**: Отправьте боту фотографию бумажного чека или скриншот банковского перевода. Gemini Vision распознает все позиции, извлечёт суммы и разнесёт их по категориям.
- 💬 **Режим чата с ИИ**: Полноценный встроенный чат-бот на базе Gemini с сохранением контекста диалога.
- 📉 **Диаграммы**: Генерация красивых столбчатых и круговых графиков (По категориям, По дням) прямо в чате с использованием Matplotlib.
- 💰 **Бюджеты**: Установка месячных лимитов на разные категории. Бот предупредит, если вы превысили лимит.
- 🔒 **Приватность**: Доступ к боту только для разрешенных пользователей (Whitelist).
- 🗄 **Локальная БД**: Использование SQLite для хранения трат (база хранится локально на вашем устройстве, никакие сторонние базы данных не требуются).

## 🚀 Установка и запуск

### 1. Требования
- Python 3.9+
- Получить токен Telegram-бота у [@BotFather](https://t.me/BotFather)
- Получить API-ключ Google Gemini в [Google AI Studio](https://aistudio.google.com/)

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

*(Основные зависимости: aiogram, google-generativeai, python-dotenv, matplotlib)*

### 3. Настройка окружения
Создайте файл `.env` в корневой папке проекта и добавьте следующие переменные:
```ini
BOT_TOKEN=ваш_токен_от_ботфазера
GEMINI_API_KEY=ваш_ключ_gemini
ALLOWED_USERS=123456789,987654321
```
*(ID пользователей можно узнать у бота @userinfobot)*

### 4. Запуск (Локально)
```bash
python bot.py
```

## 🌍 Развертывание на сервере (Ubuntu / Linux)

Есть два удобных способа запустить бота на сервере так, чтобы он работал фоном 24/7.

### Способ 1: С помощью Docker (Рекомендуемый)
Это самый простой и надёжный способ. БД сохраняется благодаря volumes.

1. Клонируйте репозиторий на сервер:
   ```bash
   git clone https://github.com/Timobzdecc/expense-tracker-bot.git
   cd expense-tracker-bot
   ```
2. Создайте и заполните `.env` файл.
3. Запустите контейнер в фоне:
   ```bash
   docker-compose up -d --build
   ```
*Для просмотра логов:* `docker-compose logs -f`
*Для остановки:* `docker-compose down`

### Способ 2: В виде службы Systemd (Без Docker)
Если вы не используете Docker, настройте бота как системный сервис.

1. Клонируйте репо, установите `python3-venv` и зависимости:
   ```bash
   git clone https://github.com/Timobzdecc/expense-tracker-bot.git /opt/expense-tracker-bot
   cd /opt/expense-tracker-bot
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. Создайте файл конфигурации сервиса:
   ```bash
   sudo nano /etc/systemd/system/expensebot.service
   ```
   Вставьте туда:
   ```ini
   [Unit]
   Description=Expense Tracker Telegram Bot
   After=network.target

   [Service]
   User=root
   WorkingDirectory=/opt/expense-tracker-bot
   ExecStart=/opt/expense-tracker-bot/venv/bin/python bot.py
   Restart=always

   [Install]
   WantedBy=multi-user.target
   ```
3. Запустите сервис:
   ```bash
   sudo systemctl daemon-reload
   sudo systemctl enable expensebot
   sudo systemctl start expensebot
   ```
*Для просмотра логов:* `sudo journalctl -u expensebot -f`

## 📖 Как пользоваться (Мануал)

### Добавление трат
- **Текстом:** Напишите сумму и описание. Порядок не важен. 
  *Пример:* `Кофе старбакс 350` или `350 кофе`.
- **Фотографией:** Скиньте фото чека. Бот обработает его 10-20 секунд и запишет все купленные позиции отдельными тратами.
- **Ошибки AI:** Если нейросеть заблокирует запрос (сработает safety-фильтр), трата всё равно будет сохранена в категорию «🔧 Прочее», так как суммы и описания парсятся надежным регулярным выражением.

### Меню
- **📊 Статистика**: Сводка за сегодня, неделю, месяц или всё время.
- **📈 По дням**: Расходы за последние 7 дней в виде текстового графика.
- **📉 Диаграммы**: Графики в виде картинок (круговая по категориям, столбчатая по дням).
- **📋 История**: Просмотр и удаление последних 5 трат.
- **💰 Бюджеты**: Установка лимита на категорию (например, 10 000 на продукты в месяц).
- **💬 Чат с ИИ**: Переход в режим диалога с умным ассистентом (выход по кнопке «🚪 Выйти из чата»).

## 🛠 Технический стек

- **Python 3**
- **aiogram 3.x** — асинхронный фреймворк для Telegram API.
- **google-generativeai** — интеграция с Gemini (модели `gemini-1.5-flash` или `gemini-1.5-pro`).
- **SQLite3** — хранение трат и лимитов (база создается автоматически — `expenses.db`).
- **Matplotlib** — отрисовка графиков (`charts.py`).
